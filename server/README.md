## Installation

```bash
$ npm install
```

## Running the app

As we use Firebase functions, local start should be with Firebase

```bash
# development
$ npm run serve

#or in watch mode
$ yarn serve:watch


# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
